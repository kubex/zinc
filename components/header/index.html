Header

<zn-header> | ZnHeader
Since 1.0 experimental

Short summary of the component’s intended use.

Dashboard Another And another 

<div style="background-color: #f5f5f5; padding: 20px;">
  <zn-header caption="John Jones"
             description="This is the Description of the header"
             previous-path="#"
             navigation="[{&quot;path&quot;:&quot;#&quot;,&quot;title&quot;:&quot;Dashboard&quot;},{&quot;path&quot;:&quot;#&quot;,&quot;title&quot;:&quot;Another&quot;}]">
    <zn-icon round src="test1@example.com" size="36"></zn-icon>
    <a href="#" slot="breadcrumb">Dashboard</a>
    <a href="#" slot="breadcrumb">Another</a>
    <a href="#" slot="breadcrumb">And another</a>
  </zn-header>
</div>

Importing

If you’re using the autoloader or the traditional loader, you can ignore this section. Otherwise, feel free to use any of the following snippets to cherry pick this component.

To import this component from the CDN using a script tag: 

<script type="module" src="https://cdn.jsdelivr.net/npm/@kubex/zinc@1.0.4/dist/components/header/header.js"></script>

To import this component from the CDN using a JavaScript import: 

import 'https://cdn.jsdelivr.net/npm/@kubex/zinc@1.0.4/dist/components/header/header.js';

To import this component using a bundler:

import '@kubex/zinc/dist/components/header/header.js';
