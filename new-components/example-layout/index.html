Example Layout

Most pages should be scaffolded in the following way. This layout is designed to be responsive and to work well on all devices. It is also designed to be easy to read and understand.

zn-sp allows you to add space between elements. It is a shorthand for zn-space. It is a flex container that has a default gap of --zn-spacing-medium. by specifying no-gap you can remove the gap between elements. So that if the component you are using already includes padding or margin it will not be doubled up.
